# Steganography-Based-Chat-App

clone the repo
```
git clone https://github.com/abdelrahman-osama/Steganography-Based-Chat-App.git
```

Install crypto
```
pip install crypto
```
Run the server
```
python PServer.py
```

Run the client 
```
python PClient.py
```

Follow the on scren instructions in the client terminal.
